## Component Overview

### The Council of Agents

1. **The Accountant** - Quantitative analysis and financial health checks
2. **The Scout** - Qualitative research and sentiment analysis  
3. **The Governor** - Risk management and allocation decisions

### Key Features

- Zero operational cost design
- Human-in-the-loop decision making
- Local data storage and processing
- Leveraged portfolio management
- Real-time risk monitoring
