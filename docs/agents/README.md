# The Council of Agents

## Overview

The Council consists of three specialized AI agents that work together to analyze investment opportunities and make portfolio management decisions.

## The Accountant

**Role**: Quantitative Analyst

### Responsibilities
- Balance sheet analysis
- Financial ratio calculations
- Debt service coverage assessment
- Valuation metrics (P/E, P/B, EV/EBITDA)
- Trend analysis of key financial indicators

### Key Protocols
- **Debt Service Protocol**: Automatically rejects companies with Debt-to-Equity > 0.5x
- **Quality Filters**: Screens for consistent profitability and cash flow generation
- **Valuation Guards**: Prevents overpaying during market euphoria

## The Scout

**Role**: Qualitative Research Analyst

### Responsibilities
- Annual report analysis using Gemini Pro
- Management commentary and guidance assessment
- Industry trend identification
- Governance risk detection
- News sentiment analysis

### Key Capabilities
- PDF document processing and summarization
- Earnings call transcript analysis
- Regulatory filing monitoring
- ESG risk assessment

## The Governor

**Role**: Risk & Portfolio Manager

### Responsibilities
- Position sizing based on conviction scores
- Portfolio rebalancing decisions
- Cash flow management for debt service
- Core-Satellite strategy implementation
- Risk limit enforcement

### Key Functions
- **Conviction Multiplier**: Adjusts position sizes based on agent consensus
- **STP Optimization**: Dynamic systematic transfer plans based on valuations
- **Liquidity Management**: Maintains operational reserves for debt payments
