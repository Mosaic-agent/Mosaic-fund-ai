# User Guide

## Getting Started with Mosaic

### Overview
Mosaic is your AI-powered fund manager that helps you make disciplined investment decisions while managing leveraged capital responsibly.

## Initial Setup

### 1. Define Your Investment Universe
- Create a watchlist of stocks you're interested in
- Set your risk tolerance and return expectations  
- Define sector preferences and restrictions
- Set maximum position sizes and concentration limits

### 2. Configure The Council
- **The Accountant**: Set financial health criteria
- **The Scout**: Configure news sources and sentiment analysis
- **The Governor**: Define risk parameters and allocation rules

### 3. Link Your Trading Account
- Connect your zero-brokerage trading account
- Verify API access and permissions
- Set up automated trade execution (optional)
- Configure trade confirmation preferences

## Daily Workflow

### Morning Routine
1. **Portfolio Review**: Check overnight news and price movements
2. **Agent Briefing**: Review The Council's overnight analysis
3. **Action Items**: Review recommended trades and adjustments
4. **Risk Check**: Verify all positions are within risk limits

### Investment Process
1. **Research**: Add new stocks to watchlist for analysis
2. **Analysis**: Let The Council evaluate fundamentals and sentiment
3. **Decision**: Review agent recommendations and conviction scores
4. **Execution**: Approve and execute recommended trades
5. **Monitoring**: Track position performance and risk metrics

## Understanding Agent Recommendations

### Conviction Scores
- **High (8-10)**: Strong buy recommendation with larger position size
- **Medium (5-7)**: Moderate conviction, standard position size
- **Low (1-4)**: Weak signal, minimal or no position
- **Veto (0)**: Fundamental concerns, avoid investment

### Trade Types
- **Core Holdings**: Large-cap ETFs for stability (40-60% allocation)
- **Satellite Positions**: Individual stocks for alpha generation (30-50%)
- **Cash Management**: Liquid funds for debt service (10-20%)

## Managing Leveraged Capital

### Debt Service Priority
- Ensure 3-month interest coverage is always maintained
- Monitor debt service coverage ratio weekly
- Set up automatic transfers to liquid funds
- Plan for early loan repayment when profitable

### Risk Monitoring
- Track portfolio-level drawdown daily
- Monitor individual position performance
- Review leverage ratios and margin requirements
- Maintain emergency liquidation plans

## Troubleshooting

### Common Issues
- API connection problems
- Data feed interruptions  
- Agent disagreement resolution
- Trade execution failures

### Getting Help
- Check system logs for error messages
- Review API status pages
- Consult documentation for configuration issues
- Report bugs through GitHub issues
